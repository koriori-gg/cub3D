## 💡 About the project
Creating an FPS game using ray casting.
## 🛠️ Usage
### Requirements
- Linux
- X Window System

### Instructions
```
git clone git@github.com:koriori-gg/cub3D.git
cd cub3D
make
./cub3D maps/OK/subject.cub
```
### Controls
| Key | Action |
| :--- | :---: |
| `W` `↓`| UP |
| `S` `↑`| DOWN |
| `D` | RIGHT |
| `A` | LEFT |
| `→` | TURN RIGHT |
| `←` | TURN LEFT |
